# TRAIT slope table

OLS slope (β) of Big Five TRAIT score vs. LoRA scale.

**How to read:** each value is the change in trait score (on the 0–1 TRAIT scale)
per +1.0x increase in LoRA adapter scaling. For example, β = +0.080 means the
trait score rises by 0.08 for every unit of LoRA scale applied.

Computed via ordinary least-squares regression across 7 scale points
(−1.5, −1.0, −0.5, 0, +0.5, +1.0, +1.5), where scale 0 = base model (no adapter).

Generated by `scripts.experiments.personality_evals.plot_hf_personas`
from eval results in `scratch/evals/personality/eval_hf-personas_<persona>/`.
