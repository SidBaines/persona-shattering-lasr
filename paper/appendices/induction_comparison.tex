%       Base                                          → ext -0.91

%   - Takeaway: weight-space and activation-space methods hit a *judge-floor*

%     give the model no contextual reason to be terse and withdrawn*.
%     Sysprompt overrides this by giving the assistant an explicit instruction.
%   - Mechanistic interpretation: system prompts and LoRAs intervene at
%     different representational levels. Sysprompt operates on the assistant's
%     response *plan* (interpreted as instruction at planning time); LoRA
%     tweaks per-token sampling distributions. Introversion as judged is
%     largely about *response shape* (length, willingness to elaborate) rather
%     than *word choice*, which is why sysprompt reaches it and LoRA doesn't.
%   - This is a real limitation we should be honest about. It's also a
%     mechanistic finding — different intervention modalities are not
%     interchangeable for arbitrary trait directions.
