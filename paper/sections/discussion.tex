\section{Discussion}\label{sec:discussion}

Limitations:

\begin{itemize}
  \item We manually looked at OCEAN. Maybe there is a more principled way of finding a huge number of some pre-defined traits we’d expect to see?
  \item Did we try a broad enough range of models
  \item Did we try a broad enough range of inputs when generating trajectories
  \item Did we run a broad and thorough enough range of evals
  \item Could we have done more work to combine suppressors and amplifiers into one direction
  \item Similarly for different traits that might lie on the same axis, were we right to treat goodness/badness separately?
  \item Did we lower the rank too much?
  \item Related, are we missing more subtle behaviours (maybe dangerous behvaiours) that didn’t make our obvious personas
  \item Did we run long enough trajectories when generating the text for fine-tuning our LoRas
\end{itemize}
